# Implementation of Hopfield Networks Using a Spiking Neuron Model
## Execution
### 1. move to the directory under which julia code exist

```bash
cd original_network/code
```
or
```bash
cd symmetrySTDP/code
```

### 2. open REPL
```bash
julia 
```

### 3. run
```jl
include("5x5.jl")
```

or 

```jl
include("mnist.jl")
```
